Introduction to Regular Expressions (Regex)

Regular expression also known as Regex is a tool for describing patterns in strings. While R provides a set of built-in functions for regex pattern matching, I recommend that you use functions from "stringr" (part of the tidyverse), which in turn is based on "stringi".

Main "stringr" functions for regex

The goal of this lab is for you to get a first contact with the most common regex functions in "stringr" (see below), and to play a little bit with matching basic regex patterns (e.g. literal characters, escaping metacharacters, using character sets, character classes, anchors, and quantifiers).

  • str_match()
  • str_detect()
  • str_extract()
  • str_locate()
  • str_sub()
  • str_split()
  • str_remove()
  • str_replace()

Some common regex patterns

Here’s a table with some of the common regex patterns:

Lieteral Characters, Wildcard, and Anchors

Pattern Description
abc letters
123 digits
. any chracter
\\. literal dot
^a starts with a
b$ ends with b

Character Sets and Ranges

Pattern Description
[abd] only a, b, or c
[^abc] not a, b, nor c
[a-z] characters a to z
[A-Z] characters A to Z
[0-9] numbers 0 to 9

POSIX Classes

Pattern Description
[[:digit:]] any digit
[[:alpha:]] any letter
[[:alnum:]] any alpha-numeric
[[:upper:]] any lower case
[[:lower:]] any upper case
[[:punct:]] any punctuation
[[:space:]] any space
[[:xdigit:]] any hexadecimal digit

Abbreviated Character classes

Pattern Description
\\d any digit
\\D any Non-digit
\\w any alphanumeric character
\\W any non-alphanumeric character
\\s any whitespace
\\S any Non-whitespace

Repetition and Quantifiers

Pattern Description
a{m} m repetitions of "a"
a{m,n} m to n repetitions of "a"
a* zero or more repetitions of "a"
a+ one or more repetitions of "a"
a? optional character "a"
